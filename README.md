# Zenn
[zenn - かいなり](https://zenn.dev/kainari)  

# Zenn CLI

## CLIをインストール
```bash
npm install zenn-cli
```

## CLIをアップデート
```bash
npm install zenn-cli@latest
```

## 記事の新規作成
```bash
npx zenn new:article --slug [slug]
```

## プレビュー
```bash
npx zenn preview
```
